filesize_format()

Output this number as easily readable filesize. Usage: 

  100_000.filesize_format             # => "97.7K"
  100_000_000.filesize_format         # => "95.4M"
  100_000_000_000.filesize_format     # => "93.1G"
  100_000_000_000_000.filesize_format # => "90.9T"
